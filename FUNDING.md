# Support the Project

If you find this project useful and would like to support its development, there are several ways to contribute:

## Direct Support

- **GitHub Sponsors** - Sponsor this project on [GitHub](https://github.com/ejm/go_pi)
- **Open Collective** - Support through [Open Collective](https://opencollective.com) (if available)

## Contribute

- **Report Bugs** - Help us find and fix issues by reporting bugs responsibly (see [SECURITY.md](SECURITY.md))
- **Suggest Features** - Share ideas for new features and improvements
- **Submit Pull Requests** - Contribute code improvements (see [CONTRIBUTING.md](CONTRIBUTING.md))
- **Improve Documentation** - Help improve guides and examples
- **Share Your Feedback** - Let us know what works well and what could be better

## Feedback

Your feedback helps make this project better. Whether it's a suggestion, question, or issue, we'd love to hear from you:

- Open a [GitHub Issue](https://github.com/ejm/go_pi/issues)
- Start a [GitHub Discussion](https://github.com/ejm/go_pi/discussions)
- Contact the maintainer via the information in [SECURITY.md](SECURITY.md)

## Thank You

Thank you for using and supporting this project!
